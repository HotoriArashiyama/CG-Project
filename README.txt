wsad ctrl space：控制camera前后左右上下，鼠标拖动改变转移镜头朝向
GUI：窗口主页，点击代表各个图形的按键后可以再在空白处点击鼠标，会在镜头中央正视的直线与平面y=0（水平面）交点处生成所选择的图形，准备添加状态下尚未点击添加则可以通过esc取消或点击其他图形按键切换要添加的图形。
滑块XYZ：待其他组员进行整合，可自定义参数，如光源位置等
鼠标点击图形：图形变为现况填充并变白色，imgui窗口变为相应图形的名称，并切换到选择窗口，可以对当前图形进行参数调整
选择状态下：
      方向键前后左右：以camera视角为基准对当前图形进行水平面上平移
     ，。（逗号，句号键）：对当前图形进行旋转
     imgui滑块：待其他组员进行整合，可自定义参数
     delete：删除当前图形并回到GUI主页
     esc：回到主页
     点击空白处：回到主页
项目中可以添加的是sphere cube ellipse和cone，#define am为项目绘制图形类型总数，如当前为4。整合时可以修改列表替换这些图形，同时添加代表各图形的按键等
